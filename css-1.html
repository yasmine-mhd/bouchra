أسئلة وأجوبة QCM - HTML و CSS

1. ما معنى HTML؟

a) HyperText Markup Language

b) HighText Machine Language

c) Hyperlink and Text Markup Language

الإجابة الصحيحة: a) HyperText Markup Language

2. ما هي الوسمة الصحيحة لإدراج صورة في صفحة HTML؟

a) <img>

b) <image>

c) <picture>

الإجابة الصحيحة: a) <img>

3. أي من الخصائص التالية تُستخدم لتغيير لون النص في CSS؟

a) background-color

b) font-color

c) color

الإجابة الصحيحة: c) color

4. أين تكتب أكواد CSS في ملف HTML؟

a) داخل وسم <body>

b) داخل وسم <style>

c) داخل وسم <script>

الإجابة الصحيحة: b) داخل وسم <style>

5. ما هي الخاصية التي تُستخدم لتغيير حجم الخط في CSS؟

a) text-size

b) font-size

c) size-font

الإجابة الصحيحة: b) font-size

6. ما هي وسيلة الربط بين ملف HTML وملف CSS خارجي؟

a) <link rel="stylesheet" href="style.css">

b) <css src="style.css">

c) <style src="style.css">

الإجابة الصحيحة: a) <link rel="stylesheet" href="style.css">

7. ما هو العنصر المستخدم لإنشاء روابط في HTML؟

a) <link>

b) <a>

c) <href>

الإجابة الصحيحة: b) <a>

8. ما هي الخاصية التي تجعل العنصر في وسط الصفحة أفقياً باستخدام CSS؟

a) align: center

b) text-align: center

c) margin: auto

الإجابة الصحيحة: c) margin: auto
