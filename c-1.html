أسئلة QCM - درس لغة C

1. ما هي الكلمة المفتاحية في C لتعريف متغير من نوع صحيح؟

a) integer

b) int

c) number

الإجابة الصحيحة: b) int

2. كيف يتم إدخال بيانات من المستخدم في C؟

a) printf()

b) scan()

c) scanf()

الإجابة الصحيحة: c) scanf()

3. أي من الخيارات التالية هو النوع الصحيح لتعريف متغير يحتوي على قيمة عشرية؟

a) int

b) float

c) decimal

الإجابة الصحيحة: b) float

4. كيف يتم تعريف مصفوفة في C تحتوي على 5 عناصر؟

a) int array[5];

b) int[5] array;

c) array[5] int;

الإجابة الصحيحة: a) int array[5];

5. ما هو الأمر المستخدم لكتابة النص إلى الشاشة في C؟

a) cout

b) print

c) printf

الإجابة الصحيحة: c) printf

6. ماذا يحدث إذا لم يتم إغلاق قوس } في برنامج C؟

a) لا يحدث شيء

b) يظهر خطأ في البرنامج

c) يتم إغلاق القوس تلقائيًا

الإجابة الصحيحة: b) يظهر خطأ في البرنامج

7. ما هي قيمة المتغير x بعد تنفيذ السطر التالي؟

int x = 5;
  x += 3;

a) 5

b) 8

c) 3

الإجابة الصحيحة: b) 8

8. كيف يتم تعريف دالة في C؟

a) def functionName()

b) function functionName()

c) void functionName()

الإجابة الصحيحة: c) void functionName()
