أسئلة QCM - SQL

1. ما هو الأمر في SQL لإضافة سجل جديد إلى جدول؟

a) INSERT INTO

b) ADD INTO

c) INSERT RECORD

الإجابة الصحيحة: a) INSERT INTO

2. ما هو الأمر في SQL لاختيار كل البيانات من جدول معين؟

a) SELECT * FROM table_name;

b) SHOW ALL FROM table_name;

c) GET ALL FROM table_name;

الإجابة الصحيحة: a) SELECT * FROM table_name;

3. أي من التالي هو نوع بيانات صحيح في SQL لتخزين نص؟

a) TEXT

b) STRING

c) CHARACTER

الإجابة الصحيحة: a) TEXT

4. ماذا تفعل الجملة WHERE في SQL؟

a) تحدد جدول البيانات

b) تحدد العمود
